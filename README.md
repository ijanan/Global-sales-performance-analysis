# Global Sales Performance Dashboard

## Overview

This project analyzes sales performance across six countries: Canada, China, India, Nigeria, the UK, and the US.

The goal is to transform raw sales transaction data into actionable business insights through data cleaning, modeling, KPI development, and interactive dashboard reporting using PostgreSQL, SQL, Power BI, and DAX.

---

## Tools & Technologies

- PostgreSQL
- SQL
- Power BI
- DAX
- Data Modeling
- Data Visualization

---

## Project Workflow

1. Imported six country-level sales datasets.
2. Consolidated the data into a centralized PostgreSQL database.
3. Cleaned and validated the data using SQL.
4. Designed a Star Schema data model.
5. Created DAX measures and KPIs.
6. Built interactive Power BI dashboards.
7. Generated business insights and recommendations.

---

## Key KPIs

- Total Revenue
- Total Profit
- Total Orders
- Profit Margin
- Revenue Growth %

---

## Key Insights

- Identified top-performing countries.
- Evaluated category-level performance.
- Analyzed product-level revenue contribution.
- Monitored monthly revenue trends.
- Generated executive-level recommendations.
